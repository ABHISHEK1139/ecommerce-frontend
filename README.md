# Ecommerce Frontend

A responsive ecommerce frontend built with React and Vite. Provides a modern shopping UI with product listings, cart functionality, and user interface components.

## Tech Stack

- **Framework**: React
- **Build Tool**: Vite
- **Language**: JavaScript
- **Linting**: ESLint

## Project Structure

```
├── public/          # Static assets
├── src/             # React components, pages, hooks
├── index.html       # Entry HTML
├── vite.config.js   # Vite configuration
└── package.json     # Project dependencies
```

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will run at `http://localhost:5173`.

## About

Frontend for a full-stack ecommerce platform developed at KL University. Consumes REST APIs from the [ecommerce-back](https://github.com/ABHISHEK1139/ecommerce-back) Spring Boot backend.
